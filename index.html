Trying out the famo.us engine

Try the Demo View on GitHub Download .zip Download .tar.gz

Installation

To install npm packages and build JavaScript bundle (using Browserify):

npm install

Development

To start a dev server:

npm start

Now the dev server should be running on localhost:1618.

Testing

To check the JavaScript code for errors:

npm test
